## Getting started

Make sure to have installed:

- [Node 20 or higher](https://nodejs.org/en) - For Prisma
- [Bun](https://bun.sh/)

Clone/fork this repo.

Install the dependencies:

```bash
bun install
```

Run the API:

```bash
bun run dev --filter=taco-api
```
